# 리스트

```python
list=[]
```

## 특징
+ 순서보장, 중복허용한다. `대괄호 사용`
+ 수정이 가능하다

### 맨 끝의 값이 -1이다.

###  삽입
#### `append()`: 마지막에 집어넣을 수있다.
#### `insert()`: 원래 값을 넣으면 그 다음 값들은 밀려난다.
`extend()`: 원하는곳에 집어넣을 수있다.

###  삭제
#### `remove()`: () 안에 있는 값들을 다 제거한다.
`pop()`: 마지막꺼 삭제, `clear()`

# 딕셔너리 

```python
d={key,all}
```

### 특징

+ 순서보장
+ 인덱스 키값을 보장
+ 중복 허용안함
+ 수정가능
+ 추가 삭제 가능

재고정리와 같다
키와 값을 쌍으로 지정한다.

#  정리

+ 여러값들을 순서대로 관리: `리스트`

+ 값이 바뀔일이 없거나 바뀌면 안된다 `튜플`

+ 값의 존재 여부가 중요하다, 중복은 안된다 `세트`

+ key를 통해 효츌적으로 데이터 관리  : `딕셔너리`(수행평가)
