# 리스트

+ 사용법

```python
list=[]
```

## 특징
+ 순서보장, 중복허용한다. `대괄호 사용`
+ 수정이 가능하다

#### 맨 끝의 값이 -1이다.

###  삽입
#### `append()`: 마지막에 집어넣을 수있다.
#### `insert(),extend()`: 원하는곳에 집어넣을 수있다.

###  삭제
#### `remove()` ,`pop()`, `clear()`
